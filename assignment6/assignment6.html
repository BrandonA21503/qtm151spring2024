(c) Regression

In this exercise you are asked to plot acceleration vs. weight and to plot
the best fit line.

  • Run a linear regression between acceleration and weight.
  • Extract the intercept and coefficient.
  • Using your predicted coefficients, compute the predicted value of acceleration
  • Plot a scatter plot with acceleration on the y axis and weight on the x axis.
  • On that same figure, plot a line with the predicted values of acceleration

Do not forget to label your axes and title.

In [ ]:
acceleration = features["acceleration"]
weight = features["weight"]


data = {'acceleration': acceleration, 'weight': weight}
df = pd.DataFrame(data)


model = smf.ols(formula='acceleration ~ weight', data=df).fit()


intercept = model.params['Intercept']
coefficient = model.params['weight']

predicted_acceleration = model.predict(df['weight'])


plt.scatter(x=weight, y=acceleration, label='Actual Data')

plt.plot(weight, predicted_acceleration, color='red', label=f'Best Fit Line\ny = {coefficient:.2f}x + {intercept:.2f}')

plt.xlabel('Weight')
plt.ylabel('Acceleration')
plt.title('Scatter Plot with Best Fit Line')
plt.legend()


plt.show()
